
# Charakter-Matching-Tool

Ein kleines Web-Tool, um Charaktere für Pen-&-Paper-Runden zu verwalten und zufällig zu matchen.

## Features
- Charaktere hinzufügen, entfernen und als Liste verwalten
- Charakterlisten als JSON-Datei speichern und laden
- Zufälliges Matching von Charakteren mit einstellbarer Matching-Anzahl
- Matching-Ergebnisse in die Zwischenablage kopieren

## Nutzung
1. Öffne die `index.html` im Browser.
2. Charaktere hinzufügen und verwalten.
3. Optional: Charaktere als Datei speichern oder laden.
4. Anzahl der Matchings pro Charakter einstellen und Matching erzeugen.
5. Ergebnisse kopieren oder speichern.

## Technologie
- HTML, CSS, JavaScript (keine externen Abhängigkeiten)
